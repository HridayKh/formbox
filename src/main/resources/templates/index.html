How it works

One form attribute. That's the whole integration.

Point any HTML form at your Formbox endpoint, submit it, and watch the request land below — exactly what happens in production.

your-site/contact.html

HTML 
<form action="https://formbox.dev/f/8k2p9q"
      method="POST">

  <input type="email"
         name="email" required>

  <textarea name="message"></textarea>

  <!-- invisible honeypot, no JS needed -->
  <input type="text"
         name="_gotcha"
         style="display:none">

  <button type="submit">Send</button>
</form>

Features

Everything a form backend should do, and nothing it shouldn't.

Formbox handles the parts of form handling nobody wants to rebuild — spam, storage, and notifications — so you can ship the rest of your product.

Plain HTML forms

Set the action attribute to your endpoint URL. No SDK, no build step, no JavaScript required.

AJAX & JSON endpoints

Submit with fetch and Formbox responds in JSON, ready for any modern frontend stack.

Built-in honeypot

An invisible trap field filters out bots before they ever touch your inbox — zero configuration required.

Discord notifications

New submissions route straight to a channel of your choice, batched at most once a minute so it stays readable.

One-click CSV export

Pull every submission into a spreadsheet whenever you need it — no API call, no waiting on a data export job.

Custom redirect URLs

Send people to a thank-you page, a pricing page, or wherever your funnel needs them to land next.

Cloudflare-powered edge

Every endpoint runs behind enterprise-grade DDoS mitigation and edge security by default — no setup needed.
